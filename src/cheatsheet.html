Модальные окна

Для вызова модальных окон подключен плагин fancybox. Для вызывающего элемента необходимо добавить класс .js-modal. Если это ссылка, то вызываемый адрес пишет в атрибуте href, если какой-то другой элемент, то прописывается атрибут data-src="%target%"

Примеры вызова окон можно посмотреть на странице system.html

Валидация форм

Для валидации форм подключен плагин jQuery Validate plugin

Для валидации формы необходимо прописать ей класс .js-validate. Правила валидации и текст сообщений об ошибке прописывается в дата-атрибуты. Пример на официальном сайте

Маски ввода в текстовые поля

Для данного функционала подключен плагин jQuery Input Mask

Есть возможность прописывать маски через дата-атрибут data-mask
